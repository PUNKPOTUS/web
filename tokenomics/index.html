Tokenomics of punkPOTUS

Tokenomics

Overview

The Punk POTUS On-chain NFT for Kamala Harris is a unique token designed to commemorate a historic political figure. This NFT collection features a fully on-chain design, with all components stored within a single, immutable smart contract.

Key Features

  • Fully on-chain: Artwork and metadata are stored on the blockchain
  • Immutable and indestructible design
  • Single smart contract houses all token components

Traits and Variations

  • 360 color schemes
  • 8 eyewear traits
  • 7 shirt logo traits
  • Special trait: Laser eyes (1% chance)

Randomization Mechanism

Trait assignment is truly random. Random number generated from a seed. Seed value: Number of popular votes in the 2024 US Presidential Election.

Activation Condition

The seed will be set in the contract after vote certification. If Kamala Harris does not win, the seed remains unset. Default image displays if the seed is not set.

Uniqueness

This token represents a novel approach to political memorabilia, combining blockchain technology with current events to create a unique, verifiable digital asset.
